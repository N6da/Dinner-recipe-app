#  Dinner Recipe App

A responsive web application that helps users discover delicious dinner recipes. Users can search for meals and view detailed recipe information, including ingredients and cooking instructions.

##  Features
- Search for recipes by name
- View recipe details
- Responsive design for all devices
- Fast and user-friendly interface

##  Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
- TheMealDB API
